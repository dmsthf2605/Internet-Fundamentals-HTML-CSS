HTML5

HTML5 new form attributes

HTML5 now introduces a new form attribute that allows you to associate any orphaned form control with any form element on the page such as placeholder, autofocus, autocomplete, required, pattern, list, multiple, novalidate, formnovalidate, form, formaction, formenctype, formmethod, formtarget.


How to set autofocus

<input type="text" name="entry01" id="entry01" autofocus="autofocus"/>

How to set a placeholder

<input type="text" name="entry02" id="entry02" placeholder="Default text"/>

How to set required for an entry

<input type="text" name="entry03" id="entry03" required="required"/>

How to create a list/datalist

<input type="text" id="entry04" name="entry04" list="fruits" placeholder="Select one"/>
<datalist id="fruits">
<option value="Apple">Apple</option>
<option value="Banana">Banana</option>
<option value="Blueberry">Blueberry</option>
<option value="Kiwi">Kiwi</option>
<option value="Mango">Mango</option>
</datalist>

How to ensure a valid email address format

<input type="email" name="entry05" id="entry05"/>

How to ensure a valid url format

<input type="url" name="entry06" id="entry06"/>
