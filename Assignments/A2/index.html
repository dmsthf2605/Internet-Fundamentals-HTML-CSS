HTML5 & CSS Normalize

HTML5 is the latest version of html and a markup language used for structuring and presenting content on the Internet. It is becoming famous because its capabilities are constantly improving. Also, it has a strong point when it comes to platform across a range of devices. Moreover, updates happen automatically for users.

Normalize.css is a small CSS file that provides better cross-browser consistency in the default styling of HTML elements. We use it because its modern, HTML5-ready, and alternative to the traditional CSS reset.
